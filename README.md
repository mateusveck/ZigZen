# ZigZen - Landing Page Responsiva

Projeto de landing page desenvolvido como prática de HTML e CSS, com foco em layout responsivo, organização visual e adaptação para diferentes tamanhos de tela.

## 📌 Sobre o projeto

A página apresenta uma landing page para o aplicativo fictício **ZigZen**, com seções de apresentação, funcionalidades, planos e preços, chamada para download e rodapé.

Este foi meu primeiro projeto aplicando **media queries** de forma mais completa, ajustando o layout para desktop, tablet e celular.

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3
- Media Queries
- Flexbox
- CSS Grid
- Variáveis CSS
- Pseudo-classes
- Gradientes
- Responsividade

## 📱 Responsividade

O projeto foi adaptado para diferentes tamanhos de tela, incluindo:

- Desktop
- Tablet
- Celular

Foram aplicados ajustes em grids, espaçamentos, tamanhos de fonte, imagens e organização das seções para melhorar a experiência em telas menores.

## 🎯 O que pratiquei neste projeto

- Estruturação semântica com HTML
- Criação de seções em uma landing page
- Uso de `display: flex`
- Uso de `display: grid`
- Criação de cards
- Estilização de botões
- Uso de variáveis CSS
- Aplicação de gradientes
- Efeitos com `hover`
- Responsividade com `@media`
- Organização de layout para mobile
